# Personal Portfolio Website

A clean, professional personal portfolio website showcasing my education, technical skills, work experience, projects, and interactive One Piece-themed data visualizations.

## Website Purpose

The purpose of this website is to display my portfolio by adding a twist to it looking like a resume. 
It also has a fun interactive data visualization page created by Javascript and SVGs


## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**

## File Structure

```
A2-Personal-Website/
├── images/             # folder for images
├── index.html           # Main homepage with resume content
├── visualizations.html  # Interactive SVG visualizations page
├── style.css           # Main styling for both pages
├── vis.css             # Additional styling for visualizations page
├── main.js             # Form validation and page interactivity
├── vis.js              # SVG visualization generation code
└── README.md           # Project documentation (this file)
```

## Features

### Main Page (index.html)
The main page features a header with my name and contact information linking to LinkedIn, GitHub, and the visualizations page. It includes sections for my education at Simon Fraser University, technical skills organized in a grid displaying programming languages, frameworks, and tools, and work experience as an IT Technician at SFU and UI/Frontend Developer Volunteer at Jitjots. The projects portfolio showcases 7+ featured projects with GitHub links. Additional sections include personal interests displayed as tag-style elements and an interactive contact form with JavaScript validation.

### Visualizations Page (visualizations.html)
The visualizations page features a Straw Hat Crew Bounties Chart, an animated horizontal bar chart displaying One Piece character bounties as a data visualization demonstrating proportional comparison. This visualization was created with JavaScript and SVG.
